### 3.3 平台资源目录（v1 新增）

#### 设计原则
- **资源与 Agent 解耦**：MCP Server、Skill、知识库作为平台级资源预先配置
- **Agent 按需选用**：创建 Agent 时从资源目录中选择，而非重复配置
- **支持工具级粒度**：选择 MCP Server 时可进一步筛选启用哪些 tools

#### mcp_server_registry（MCP Server 注册表）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PRIMARY KEY AUTOINCREMENT | |
| name | VARCHAR(100) | NOT NULL UNIQUE | Server 标识名，如 `mysql-prod` |
| display_name | VARCHAR(100) | | 显示名称 |
| description | TEXT | | 功能描述 |
| base_url | VARCHAR(500) | NOT NULL | MCP Server 地址, 如 `http://mysql-mcp.internal:3000` |
| headers | TEXT | DEFAULT '{}' | 请求头(JSON), 如 `{"Authorization":"Bearer xxx"}` |
| timeout | INTEGER | DEFAULT 30 | 请求超时(秒) |
| discovered_tools | TEXT | DEFAULT '[]' | 自动发现的工具列表(JSON) |
| status | VARCHAR(20) | DEFAULT 'active' | active/disabled/error |
| last_checked_at | TIMESTAMP | | 上次连接检查时间 |
| created_by | INTEGER | REFERENCES users(id) | |
| created_at | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP | |
| updated_at | TIMESTAMP | | |

**discovered_tools 字段说明**：平台通过 `tools/list` 自动发现 MCP Server 提供的工具，存储在此字段供 Agent 选择时参考。

```json
[
  {"name": "query_slow_log", "description": "查询慢查询日志"},
  {"name": "query_table_size", "description": "查询表大小"},
  {"name": "execute_sql", "description": "执行SQL语句（只读）"}
]
```

#### agent_mcp_links（Agent-MCP 关联表）

Agent 从资源目录中选择 MCP Server，并可按需过滤工具：

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INTEGER | PRIMARY KEY AUTOINCREMENT | |
| agent_id | INTEGER | NOT NULL REFERENCES agents(id) ON DELETE CASCADE | |
| mcp_server_id | INTEGER | NOT NULL REFERENCES mcp_server_registry(id) | |
| enabled_tools | TEXT | DEFAULT '[]' | 启用的工具名列表(JSON)，空数组=全部 |
| enabled | BOOLEAN | DEFAULT 1 | 是否启用 |

**示例**：Agent 选择了 `mysql-prod` server，但只启用查询类工具：
```json
{"enabled_tools": ["query_slow_log", "query_table_size"]}
```

#### agents 表调整

原有 `mcp_servers`、`skills` 字段保留用于 Agent 自定义配置（边缘场景），新增 `knowledge_base_ids` 字段作为便捷引用：

```sql
-- agents 表新增字段
ALTER TABLE agents ADD COLUMN knowledge_base_ids TEXT DEFAULT '[]';  -- [1, 3, 5]
```

Agent 的最终 MCP 工具 = `agent_mcp_links`（从目录选择）+ `mcp_servers`（Agent 自定义）
