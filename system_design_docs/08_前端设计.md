## 六、前端设计

### 6.1 技术栈

| 类别 | 技术 | 说明 |
|------|------|------|
| 框架 | Vue 3 + TypeScript | Composition API |
| 构建工具 | Vite | 快速开发 |
| UI组件库 | Ant Design Vue | 企业级组件 |
| 工作流画布 | Vue Flow | 拖拽式编排 |
| 实时通信 | WebSocket (native) | 执行进度实时推送 |
| 状态管理 | Pinia | 轻量状态 |
| HTTP请求 | Axios | |
| 代码编辑器 | Monaco Editor | Prompt编辑 |
| 样式 | TailwindCSS + SCSS | |

#### 6.4.3 每个菜单的展示内容

##### 📊 主控台

| 区域 | 内容 | 数据来源 |
|------|------|---------|
| 统计卡片行 | Agent 总数、工作流总数、应用总数、今日执行次数 | 各 API count 聚合 |
| 最近执行列表 | 最近 10 次工作流执行，状态 + 耗时 + 快捷跳转追踪 | `GET /monitor/traces?limit=10` |
| 资源状态概览 | MCP Server 在线/离线数，知识库同步状态 | 各资源 API status 字段 |
| 快捷入口 | 创建 Agent / 编排工作流 / 注册 MCP Server | 页面跳转 |

##### 🤖 Agent 列表 → 详情/编辑

| 区域 | 内容 |
|------|------|
| 列表页 | 搜索框 + 类型筛选(local/http/claudecode) + 状态筛选；表格列：名称、类型标签、关联资源数(MCP/KB/Skill)、状态、更新时间；操作：编辑/复制/测试/删除 |
| 编辑页 | 详见 [6.3.1 Agent编辑页面](#631-agent编辑页面资源目录选择模式)：基础信息 → Agent 类型配置 → System Prompt → 资源目录选择器(MCP+KB+Skill 勾选) → 自定义配置(可选展开) |

##### 🔀 工作流列表 → 编辑器

| 区域 | 内容 |
|------|------|
| 列表页 | 搜索框 + 类型筛选(supervisor/sequential/graph) + 状态筛选(draft/published)；表格列：名称、类型标签、Agent 数量、版本号、状态、更新时间；操作：编辑/发布/验证/删除 |
| 编辑器页 | 左侧 Agent 拖拽面板(可用 Agent 列表) + 中央 Vue Flow 画布(拖拽连线) + 右侧属性面板(选中节点时显示配置)；画布节点类型：主管Agent / 工作Agent / 条件判断 / 并行组 / 人工中断 / 结束；底部工具栏：保存/发布/验证/撤销/重做 |

##### 📱 应用列表 → 编辑

| 区域 | 内容 |
|------|------|
| 列表页 | 表格列：名称、关联工作流、API Key(脱敏+复制)、限流值、启用状态、调用次数；操作：编辑/启停/轮换密钥/删除 |
| 编辑页 | 选择关联工作流(下拉搜索) + 限流配置 + API Key 显示/轮换；下方卡片：调用统计(近 7 天调用量折线图) + 最近调用记录 |

##### 📦 资源目录

| 子菜单 | 列表页内容 | 编辑页内容 |
|--------|----------|----------|
| MCP Server 管理 | 搜索+状态筛选；卡片列表：名称、URL、工具数、状态灯(●active/●error)、上次检查时间；操作：发现工具/测试连接/编辑/删除 | 名称+URL+Headers+超时时间表单；下方自动发现的工具列表；[测试连接]按钮 |
| 知识库管理 | 搜索+类型筛选(file/url/rag)；卡片列表：名称、类型图标、文档数、状态、上次同步时间；操作：同步/上传文档/编辑/删除 | 名称+类型选择→动态表单(file选目录/url填地址/rag配参数)；文档列表+上传按钮 |
| Skill 管理 | 分类标签+标签筛选+搜索；卡片列表：名称、分类、类型、标签、引用Agent数；操作：预览/编辑/删除 | 名称+分类+标签+类型+内容编辑器(Monaco Editor 或文本域) |

#### 6.4.4 导航交互规则

| 场景 | 行为 |
|------|------|
| 侧边栏折叠 | 分组可折叠，点击分组标题展开/收起子菜单 |
| 当前页高亮 | 匹配当前路由，父菜单 + 子菜单均高亮 |
| 面包屑 | 列表页 → 编辑页 显示面包屑导航（如：Agent 管理 > 编辑 Agent） |
| 列表→编辑→返回 | 编辑保存成功后跳回列表页，并刷新列表 |
| Agent 列表→工作流编辑器 | 从 Agent 列表可直接"加入工作流"快捷跳转 |
| 工作流编辑器→应用 | 工作流发布后可一键"创建应用" |
| 应用列表→追踪 | 应用最近调用记录中可跳转对应执行追踪 |
| 权限控制 | 系统设置仅 admin 可见；资源目录的创建/编辑/删除仅 admin 可操作 |

#### 6.4.5 页面间关系

```
主控台
  ├─ [快捷入口] → Agent 列表 ──→ Agent 编辑
  │                  └──→ 工作流编辑器 (引用 Agent)
  │
  ├─ [快捷入口] → 工作流列表 ──→ 工作流编辑器
  │                  └──→ 应用编辑 (关联工作流)
  │
  ├─ [快捷入口] → MCP Server 管理
  │
  └─ [最近执行] → 执行追踪详情

资源目录 (创建 Agent 时引用)
  MCP Server 管理 ──→ Agent 编辑 (MCP 选择器)
  知识库管理    ──→ Agent 编辑 (KB 选择器)
  Skill 管理    ──→ Agent 编辑 (Skill 选择器)

监控
  执行追踪 ←── 主控台 / 应用详情 / 对话测试
  对话测试 ←── 应用列表
```
