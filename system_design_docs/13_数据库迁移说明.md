# 数据库选型与迁移说明

## 当前选型

| 版本 | 数据库 | ORM | 驱动 |
|------|--------|-----|------|
| v1 | MySQL 8.0 | Tortoise ORM >= 0.20 | aiomysql |
| 迁移工具 | Aerich | Tortoise ORM 官方 | - |

## 选型原因

| 维度 | MySQL 8.0 | SQLite (之前考虑) |
|------|-----------|-------------------|
| 并发写入 | 行级锁，高并发 | 库级锁，低并发 |
| JSON 支持 | 原生 JSON 类型+函数 | TEXT 存储，无查询 |
| 连接池 | Tortoise ORM 内置 | 单文件 |
| 运维 | 成熟生态，备份恢复 | 文件复制 |

## 选型历史

| 日期 | 变更 |
|------|------|
| 2026-01 | 初版：PostgreSQL + SQLAlchemy |
| 2026-06-09 | 轻量化：SQLite + SQLAlchemy |
| 2026-06-11 | 生产化：MySQL 8.0 + Tortoise ORM |
